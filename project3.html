Graph Sandbox

Description

This is a Graph Data Structure Visualization Desktop GUI! This project offers an interactive graphical user interface where you can explore and manipulate graph structures with ease. By utilizing this GUI, you gain the understanding of various operations on graphs, include Depth First Search, Breadth First Search, etc.
