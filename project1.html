HattApp - Workout Tracker

Description

A native android mobile application. Key features: CREATE and CUSTOMIZE exercises and workouts, VISUAL CALENDAR that allows users to schedule their workout routines, Realtime TIMER allows users to have a timer between their exercises during their workout, REPORT allows users to see their past completed or missed workout, and CLOUD SAVING allows user to use the same account from different device.

Technologies Used

  • Languages:
    • Java
    • XML
  • Framework and tools:
    • Android Studio
    • Google Authentication
    • Google Firebase
    • SQLite
